# Shieldcore
**Shieldcore is a powerful anti-cheat framework for Roblox using Typescript**

**Shieldcore is:**
- **⚡Easy to use**
- **🚀Powerful**
- **⚙️Extensible**

# Installation
1. run `npm install shieldcore`
2. go into node_modules/shieldcore/src/server
3. copy the decorators folder into your src/server
